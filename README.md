# Paleogeography files associated with DeepMIP-Eocene-Phase 2

The notebooks here are used to compare and generate a combine into a new paleogeography for use in Phase 2 of the early Eocene MIP.
Details on the experimental design for DeepMIP-Eocene-p2 can be found [here](https://egusphere.copernicus.org/preprints/2026/egusphere-2025-6135/)

Contents of the notebooks are as follows:
1. this notebook compares a number of different Eocene paleogeographies (largely, their paleotopography), e.g. relative to ETOPO.
2. this notebook combines different paleogeographies per region as well as removes major artefacts where possible. Note that all paleogeographies that are being combined are first rotated into the preferred plate tectonic model/reference frame prior to merging.
4. this notebook rotates various paleogeography reconstructions into different reference frames and/or times.
