# Readme for Muller++_2008_Science

This is a copy of paleobathymetry associated with the Muller et al. (2008) model.
They were obtained from: https://www.earthbyte.org/webdav/ftp/Data_Collections/Muller_etal_2008_Science.zip

Full reference: Müller, R.D., Sdrolias, M., Gaina, C., Steinberger, B., and Heine, C., 2008, Long-Term Sea-Level Fluctuations Driven by Ocean Basin Dynamics. Science, 319, 1357-1362, https://doi.org/10.1126/science.1151540
