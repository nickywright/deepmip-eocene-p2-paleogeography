# Readme for Paxman++_2019_P3

This folder contains the Eocene maximum extent paleotopography ('eocene_palaeotopography_max_34ma_5km.nc'), obtained from the supplementary grid files associated with Paxman et al. (2019).

Reference: 
Paxman, G.J.G., Jamieson, S.S.R., Hochmuth, K., Gohl, K., Bentley, M.J., Leitchenkov, G., and Ferraccioli. F., 2019. Reconstructions of Antarctic topography since the Eocene–Oligocene boundary. Palaeogeography, Palaeoclimatology, Palaeoecology, v535: 109346. https://doi.org/10.1016/j.palaeo.2019.109346
