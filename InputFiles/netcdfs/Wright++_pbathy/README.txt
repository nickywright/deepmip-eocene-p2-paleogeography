# Paleobathymetry

These files are created using the 'traditional' workflow, outlined in Wright et al., 2020; ESR.
They use the Zahirovic et al. 2022 plate model, in a paleomagnetic ('PMAG') reference frame (anchored plate ID = 701701), and the corresponding age grids. 
Age is converted into basement depth using the GDH1 (Stein and Stein 1992) relationship. Sediment thickness is from Dutkiewicz et al. 2017, with the isostatic correction from Sykes (1996). Large igenous provinces (LIPs) and seamounts are reconstructed based on similar methods to Müller et al. 2008, but using contours to gain more realistic seafloor topography (e.g. Cesile, 2016). LIP and seamount outlines are extracted from the oceanic realm in the Johannson et al. 2018 dataset (available as part of the GPlates GeoData).

Wright, N.M., Seton, M., Williams, S.E., Whittaker, J.M. and Müller, R.D., 2020. Sea-level fluctuations driven by changes in global ocean basin volume following supercontinent break-up. Earth-Science Reviews, 208, p.103293, https://doi.org/10.1016/j.earscirev.2020.103293
